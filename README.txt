<h4>Overview</h4>

All Signs Customizer adds a configurable sign editor to WooCommerce products.
It supports products such as acrylic signs, metal signs, aluminum signs, door signs, and plastic signs.

<h4>Features</h4>
<ul>
<li>Custom TTF fonts support</li>
<li>Optional Google Fonts support for font families selected in the plugin</li>
<li>Custom colors palette for limited colors usage</li>
<li>Definition of customization sizes and number of characters per text</li>
<li>Translation ready</li>
<li>Shortcode available</li>
<li>Design data in cart</li>
<li>Pricing calculation </li>
<li>Automatic calculation of text height </li>
<li>Responsive and user-friendly</li>
</ul>

<h4>Pro features</h4>

<p><strong>This section describes how to install the plugin and get it working</strong></p>

<h4>Automatic installation (easiest way)</h4>

<p>To do an automatic install of All Signs Customizer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>

<p>In the search field type "All Signs Customizer" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".</p>

<h4>Manual installation</h4>

<p><strong>Uploading in WordPress Dashboard</strong></p>

<ol>
<li>Download <code>all-signs-customizer-for-woocommerce.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>all-signs-customizer-for-woocommerce.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p><strong>Using FTP</strong></p>

<ol>
<li>Download <code>all-signs-customizer-for-woocommerce.zip</code></li>
<li>Extract the <code>all-signs-customizer-for-woocommerce</code> directory to your computer</li>
<li>Upload the <code>all-signs-customizer-for-woocommerce</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p>The WordPress codex contains <a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to install a WordPress plugin</a>.</p>

= Will All Signs Customizer work with my theme? =

All Signs Customizer works with any theme, including the default WordPress themes.

= Does the plugin use any external services? =

The plugin can optionally load Google Fonts font files for font families selected in the plugin settings. This happens only when a Google Font is selected in the plugin.

Service name: Google Fonts
Service URL: https://fonts.google.com/
Service domains: https://fonts.googleapis.com/ and https://fonts.gstatic.com/
Privacy policy: https://policies.google.com/privacy
Terms of service: https://policies.google.com/terms
Account required: No

= Where can I review the source code for the generated JavaScript and CSS files? =

The plugin ships with its uncompiled source code in the `src/` directory.
Production assets are generated into `assets/js/` and `assets/css/` with webpack.
Bundled browser-side third-party utilities, including the local `PDFObject` helper used for PDF previews, are stored in `assets/utilities/`.

Build workflow:

1. Run `npm install`
2. Run `npm run build`

The repository also includes the build configuration in `package.json` and `webpack.config.js`, and bundled third-party browser libraries in `assets/utilities/`.
